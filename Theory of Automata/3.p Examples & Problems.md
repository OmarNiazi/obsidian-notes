**Q: Give a recursive definition for the set ODD = {1,3,5,7...}.** 
**SOL:**
- **STEP 1:** Let 1 $\in$ ODD.
- **STEP 2:**
	- If x $\in$ ODD, then x + 2 and x - 2 $\in$ ODD. 
	- or if x $\in$ ODD, then x + k $\in$ ODD where k $\in$ EVEN.
	- **ALT:** we can also assume that L $\subset$ {1}* , meaning L is a subset of an infinite set made up of the character '1' i.e. $\{\wedge, 1, 11, 111, 1111...\}$. Then we say 1 $\in$ L. We can then say that if w is in L, then w11 $\in$ L. Then we can define the set ODD as ODD = $\{|w| : w \in L\}$ i.e. the length of any substring in L (which will always be odd)

- **STEP 3:** Nothing else is in $\Sigma$ unless it can be built by the above rules.

**Q: Design a Recursive Definition for a language where no word contains the substring "aa".**  $\Sigma = \{a,b\}$
**SOL:**
- **STEP 1:** Let a, b $\in$ $\Sigma$. 
- **STEP 2:**
	- If x $\in \Sigma$, then xb $\in \Sigma$. 
	- If x $\in \Sigma$ and x does not end with a, then xa $\in \Sigma$
- **STEP 3:** Nothing else is in $\Sigma$ unless it can be built by the above rules.
